# LieSpy - Detect AI

## Description
LieSpy is a open-source Google extension that utilizes the [SightEngine API](https://dashboard.sightengine.com/) to scan the images on a page and determine if they are AI generated or not.

## Installation
1. Download files
2. Go to Google extensions page
3. Turn on developer mode
4. Hit load unpacked in top left
5. Select the folder containing LieSpy files
6. Enjoy!

## Usage
LieSpy is used for any valid images to see if they are AI-Generated or real. It can be used on nearly any website. The current threshold is 80%.
